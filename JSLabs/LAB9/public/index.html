Список путей и их функционал

  • "/" - коренной путь, работающий как GET-запрос, в данный момент вы находитесь здесь
  • "/v1" - GET-запрос с приветствием на сервере
  • "/v1/stats" - Get-запрос, который возвращает html таблицу с полями: user-agent(информация о пользователе) и количество запросов
  • "/v1/comments" - POST-запрос, который возвращает клиенту сообщение 'Спасибо, за вашу отзывчивость!', если запрос сработал корректно.
  • "/v1/comments" - GET-запрос, который возвращает клиенту сообщения, которые было отправлено с помощь POST-запроса.
